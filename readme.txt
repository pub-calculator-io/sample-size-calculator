This sample size calculator enables you to calculate the minimum sample size and the margin of error. Learn about sample size, the margin of error, & confidence interval.

[https://www.calculator.io/sample-size-calculator/](https://www.calculator.io/sample-size-calculator/)

== Usage ==

Add the Sample Size Calculator shortcode to your page, post or sidebar:

`[ci_sample_size_calculator]`

Add the following code to your template where you would like the Sample Size Calculator to appear:

`<?php display_ci_sample_size_calculator(); ?>`

2. Activate the Sample Size Calculator plugin through the "Plugins" menu in WordPress.
